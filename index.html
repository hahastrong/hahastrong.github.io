python深度学习相关函数介绍

np.ceil 与 np.floor

np.ceil该函数是向上取整，如-1.7，则变成-1；3.4则变成4.
np.floor该函数是用于向下取整，如-1.7，变成-2；3.4变成3。

np.tile

用于将某一向量重复多次。
如np.tile(A, 2)，就是将向量A重复2次，按行叠加。

np.argsort()

用于返回数组排序后的索引，默认是从小到大。
如

x = np.array([3, 1, 2])
print(np.argsort(x))

array([1, 2, 0])

np.hstack()

在列的方向拼接矩阵
如

x = array([1, 2, 3])
y = array([4, 5, 6])
print(np.hstack((x, y)))

array([1, 2, 3, 4, 5, 6])

x = array([ [1], [2], [3] ])
y = array([ [4], [5], [6] ])
print(np.hstack((x, y)))

array([ [1,2 ],
[3,4],
[5,6]])
