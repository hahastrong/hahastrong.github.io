目标跟踪论文笔记《三》：Towards a Better Match in Siamese Network Based Visual Object Tracker (Siam-BM)

目标跟踪

算法思路

在训练网络和跟踪时，对那些目标长宽比例远大于1的添加一个mask，减少背景信息的干扰；同时，在跟踪过程中，同时预测目标的尺度和目标旋转角度的预测。

motivation

CNN网络提取的特征对目标的尺度和旋转不具有不变性，同时对那些目标长宽比例不匀的视频序列来说，CNN提取的特征占有太多的背景信息，导致对目标固有信息的偏离。

做法

角度和尺度估计

跟尺度估计一样，选取不同的角度偏转 candidate 以适应角度估计的目标。但是不能简单与尺度估计采取一一枚举的方法，本文作者采用了定一法。就是固定尺度或者角度，只改变角度偏转或者尺度大小，如下图：




如上图所示，作者选取3个尺度和3个角度。采用定一法则有5个组合$(1.0375, 0), (0.964,0), (1,0), (1, \pi/8), (1, -\pi/8)，相较于一一组合数量降低了一半，大大减少了计算量。

Thoughts

  • 针对部分目标长宽比例不匀， 导致跟踪算法无法实时获取目标的位置变化，作者添加了mask减少背景信息的干扰。针对相似目标有啥解决方案吗？ 难道在训练时采用图片数据集，增加样本的多样性和背景的复杂性，从而提升模型的分辨力？
  • 针对目标存在旋转情况，此时模型的IOU分数不高，作者添加一个角度偏转估计。这种方法对那些刚性旋转目标的跟踪性能有较大的的提升。。。但是像人那样两条腿交互位置的变化、跳水时身体的展开和蜷缩、人体转身问题似乎暂时想不到较好的解决方案！！！！
